CIE Movie App

This application features an Angular front-end that uses HTML5 and CSS (with the Bootstrap 5 framework) to create views. This front-end communicates with a C# Web API to pull a sample movie database stored in a SQL Server 2019 Database. Authentication is performed using JWT Authentication, and Entity Framework is used to model and interface with the Database. I tried to use as much abstraction as I could, and added reusability and potential for expansion throughout.
